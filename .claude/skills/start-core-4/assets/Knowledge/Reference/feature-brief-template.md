# Feature Brief Template

## Feature Name
[Clear, specific name — not a project codename]

## Problem Statement
What user problem does this solve? Who experiences it? How often? What's the cost of not solving it?

## Proposed Solution
What are we building? Be specific enough that an engineer could estimate scope but not so detailed that you're designing the UI.

## Success Metrics
How will we know this worked? Define 2-3 measurable outcomes with targets and timeframes.
